# Apit212

This is a Pyhton based API using selenium and requests. 

## Requirments
* Selenium
* Requests
* Python3
* Firefox

## Installation

pip install apit212

## Import

### Demo account: 

```
from apit212 import *

Client = Apit212(username="flock92@account.api", password="pass******", mode="demo")
````

### Live account:

```
from apit212 import *

Client = Apit212(username="flock92@account.api", password="pass******", mode="live")
````
